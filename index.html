Temple

Temple is a language engine framework which enables you to create your own translation engine for your favourite language. Its main purpose is creating a pipeline four your frontend code which uses the same variables in all languages.

It is also easy to understand and extensible trough events.

Main benefits

I stared this project because i was tired of all the template engines which where not object orientated or simple to use/extend. Now im left with a engine that provides full customization for your needs.
